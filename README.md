# ⚡ Mini Projects Hub

A futuristic, terminal-styled portfolio showcasing a collection of 18 frontend mini-projects. Built with high-end aesthetics, CRT effects, and smooth interactive logic.

## 🌌 Overview

This repository is a curated collection of web experiments ranging from basic DOM manipulations to advanced 3D WebGL visualizations. The projects are organized into a central hub designed as a **Futuristic Terminal**, complete with a boot-up sequence and system-level interactions.

## 🚀 The Hub Features

- **CRT Scanline Effects**: Authentic retro-futuristic visuals.
- **Boot Sequence**: Dynamic system initialization animation.
- **Complexity Sorting**: Projects are categorized from **Basic** to **Expert**.
- **Unified UI**: Consistent high-end design across all major modules.

## 📥 How to Run

1. Clone or download this repository.
2. Open the root `index.html` in any modern web browser.
3. Wait for the boot sequence to complete.
4. Click **> RUN_SCRIPT** on any project card to launch it.
