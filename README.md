# ⚡ Mini Projects Hub

A high-performance, futuristic terminal-styled portfolio showcasing a collection of **24 Frontend Mini-Projects**. Built with a focus on modern aesthetics, interactive logic, and advanced CSS/JS techniques.

> [!NOTE]
> This repository serves as a progression journey from basic DOM manipulation to advanced 3D WebGL visualizations.

## 🌌 Repository Overview

The core of this project is a **Futuristic Terminal Hub** (`index.html`) which acts as a centralized dashboard. It features:
- **Authentic CRT Visuals**: Scanlines, flicker, and glow effects.
- **Dynamic Boot Sequence**: A terminal-style initialization on load.
- **Interactive Scramble Header**: Cyberpunk-inspired text animations.
- **Automated Registry**: Projects are injected via `script.js` for easy management.

## 🛠️ Technical Stack

- **Core**: HTML5, Vanilla CSS3, Javascript (ES6+)
- **Animation**: GSAP (GreenSock), CSS Keyframes
- **Graphics**: Canvas API, Three.js (WebGL)
- **Design Concepts**: Glassmorphism, Neumorphism, Cyberpunk Aesthetics
- **Typography**: Orbitron, Fira Code, Inter

## 📥 Getting Started

1. **Clone the Repo**: `git clone <repo-url>`
2. **Launch the Hub**: Open `index.html` in your browser.
3. **Execute Scripts**: Navigate through the terminal and click **> RUN_SCRIPT** to launch individual experiments.
